/* 階層の概要
=======================================================================

■controller
 -　機能そのもののファイル(php)。

■css
 -topページに適応されるCSSファイル

■img
 -topページに適応される画像

■inc
 -インクルードさせるファイル

■js
 -topページに適応されるJSファイル

■login
 -ログイン画面

■new_user
 -アカウント作成画面

■pass
 -パスワード変更画面

■setting
 -セッティング画面

■start
 -新規ユーザ用ゲームの説明画面
